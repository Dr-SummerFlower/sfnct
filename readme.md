<p>用来快速创建代码模版。</p>

#### 安装

请使用`npm`或`yarn`全局安装此包

```bash
npm install -g sfnct
```

#### 创建命令

1. create命令----创建一个新的模版，需要配合其他参数执行

```bash
sfnct -c```

| 参数                   | 说明     |
|:---------------------|:-------|
| `sfnct -c -t <name>` | 创建应用模版 |
| `sfnct -c -f <name>` | 创建文件模版 |

2. list命令----列出所有模版

```bash
sfnct -l
```

3. OS命令----列出当前系统信息

```bash
sfnct -o
```

4. help命令----查看帮助信息

```bash
sfnct -h
```

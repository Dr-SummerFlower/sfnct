<p>用来快速创建代码模版。</p>

#### 安装

请使用`npm`或`yarn`全局安装此包
```bash
npm install -g sfnct
```

#### 创建命令
1. create命令
```bash
使用命令`sfnct -c`创建一个新的模版，需要配合其他参数执行
    使用命令`sfnct -c -t <name>`创建应用模版
    使用命令`sfnct -c -f <name>`创建文件模版
```
2. list命令
```bash
使用命令`sfnct -l`列出所有模版
```
3. OS命令
```bash
使用命令`sfnct -o`列出当前系统信息
```
4. help命令
```bash
使用命令`sfnct -h`查看帮助信息
```
